1. Base y dimensión#

Ejemplo de Base y dimensión en \(\mathbb{R}^3\)#

  1. Verificar si los vectores \(\mathbf{u}\), \(\mathbf{v}\) y \(\mathbf{w}\) forman una base de \(\mathbb{R}^3\)

\[\begin{split} \mathbf{u}= \begin{pmatrix} 1\\ 2\\ 0 \end{pmatrix}, \qquad \mathbf{v}= \begin{pmatrix} 0\\ 1\\ 1 \end{pmatrix}, \qquad \mathbf{w}= \begin{pmatrix} 2\\ 0\\ 1 \end{pmatrix} \end{split}\]

Usando Python podemos hacer los siguiente

  • Construir una matriz \(A\) cuyas columnas sean los vectores \(\mathbf{u}\), \(\mathbf{v}\) y \(\mathbf{w}\)

  • Usar Python para calcular el determinante de \(A\) y determinar si los vectores efectivamente forman una base del espacio \(\mathbb{R}^3\)

import numpy as np
import matplotlib.pyplot as plt

u =  np.array([1, 2, 0])
v =  np.array([0, 1, 1])
w =  np.array([2, 0, 1])

A = np.column_stack((u, v, w))

det_A = np.linalg.det(A)

print(det_A)

5.000000000000001

\(\mathbf{u}\), \(\mathbf{v}\) y \(\mathbf{w}\) son linealmente independientes y por lo tanto forman una base de \(\mathbb{R}^3\)

  1. Sea \( \mathbf{p}= \begin{pmatrix} 3\\ 4\\ 2 \end{pmatrix} \)

Exprese el vector \(\mathbf{p}\) como una combinación lineal de los vectores \(\mathbf{u}\), \(\mathbf{v}\) y \(\mathbf{w}\), es decir

\( \mathbf{p} = \alpha_1\mathbf{u} + \alpha_2\mathbf{v} + \alpha_3\mathbf{w}\)

d = np.array([3, 4, 2])

coeficientes =  np.linalg.solve(A, d)

#print(coeficientes)
print(f" p = {coeficientes[0]}u + {coeficientes[1]}v {coeficientes[2]}w")

 p = 1.4u + 1.2v 0.8w

Determine si dichos vectores forman una base de \(\mathbb{R}^2\). En caso afirmativo, exprese el vector \(\mathbf{p}=\begin{pmatrix} 7\\ 5 \end{pmatrix}\) como una combinación lineal de los vectores \(\mathbf{u}\) y \(\mathbf{v}\)

Ejemplos de espacios generados#

  1. Considere el vector

\[\begin{split} \mathbf{v_1}= \begin{pmatrix} 2\\ 1 \end{pmatrix} \end{split}\]

El espacio generado por \(\mathbf{v_1}\) está dado por

\[ \operatorname{gen}\left\{\mathbf{v_1} \right\} = \left\{\mathbf{v}: \mathbf{v} = a\mathbf{v_1}, a\in\mathbb{R} \right\} \]
v1 = np.array([2, 1])

a_values = np.linspace(-3, 3, 500)

x = []
y = []

for a in a_values:
    v = a*v1
    x.append(v[0])
    y.append(v[1])


plt.scatter(x, y, s=2, color="gray")
plt.quiver(0, 0, v1[0], v1[1],
           angles="xy",
           scale_units="xy",
           scale=1,
           color="green")

plt.text(v1[0] + 0.1 , v1[1] - 0.5 , r'(2,1)',
         fontsize=12, color='green')

plt.axis("equal")
plt.grid()
plt.show()

  1. Considere los vectores

\[\begin{split} \mathbf{w_1}= \begin{pmatrix} 1\\ 0 \end{pmatrix}, \qquad \mathbf{w_2}= \begin{pmatrix} 0\\ 1 \end{pmatrix} \end{split}\]

El espacio generado por \(\mathbf{w_1}\) y \(\mathbf{w_2}\) está dado por

\[ \operatorname{gen}\left\{\mathbf{w_1},\mathbf{w_2}\right\} = \left\{ \mathbf{w}: \mathbf{w}=a\mathbf{w_1}+b\mathbf{w_2}, \;a,b\in\mathbb{R} \right\}, \]

en donde

\[\begin{split} a\mathbf{w_1}+b\mathbf{w_2} = a \begin{pmatrix} 1\\ 0 \end{pmatrix} + b \begin{pmatrix} 0\\ 1 \end{pmatrix} = \begin{pmatrix} a\\ b \end{pmatrix} \end{split}\]
w1 = np.array([1, 0])
w2 = np.array([0, 1])

x = []
y = []

for a in np.linspace(-3, 3, 100):
    for b in np.linspace(-3, 3, 100):
        w = a*w1 + b*w2
        x.append(w[0])
        y.append(w[1])

plt.scatter(x, y, color="gray", s=1)

plt.quiver(0, 0, w1[0], w1[1],
           angles="xy",
           scale_units="xy",
           scale=1,
           color="red")

plt.quiver(0, 0, w2[0], w2[1],
           angles="xy",
           scale_units="xy",
           scale=1,
           color="blue")

plt.axis("equal")
plt.grid()
plt.show()

Considere los vectores

\[\begin{split} \mathbf{w_1}= \begin{pmatrix} 1\\ 0\\ 0 \end{pmatrix}, \qquad \mathbf{w_2}= \begin{pmatrix} 0\\ 1\\ 1 \end{pmatrix}. \end{split}\]

El espacio generado por \(\mathbf{w_1}\) y \(\mathbf{w_2}\) está dado por

\[ \operatorname{gen}\left\{\mathbf{w_1},\mathbf{w_2}\right\} = \left\{ \mathbf{w}: \mathbf{w}=a\mathbf{w_1}+b\mathbf{w_2}, \;a,b\in\mathbb{R} \right\}. \]
w1 = np.array([1, 0, 0])

ax.quiver(0, 0, 0, 0, 0, L, arrow_length_ratio=0.05, color='black')


# Etiquetas
ax.text(w1[0] + 0.1, w1[1], w1[2],
        '(1,0,0)', color='red')

ax.text(w2[0], w2[1] + 0.1, w2[2],
        '(0,1,1)', color='blue')

# Ejes
ax.set_xlabel('x')

ax.set_zlabel('z')

2. Números complejos#

Los números complejos en Python se pueden declarar usando la función complex() o directamente utilizando la notación \(a + bj\).

Importante: La unidad imaginaria en Python se denota con \(j\), i.e. \(j^2=-1\)

Ejemplo

z =  1 + 2j

w = complex(1, 2)

print(z)
print(w)
print(z == w)

(1+2j)
(1+2j)
True

Conversión de flotantes/enteros a complejos#

var1 =  1
var2 = np.pi

complejo1 = complex(var1)
complejo2 = complex(var2)

print(complejo1, complejo2)

(1+0j) (3.141592653589793+0j)

Para acceder a la parte real e imaginaria de un número complejo utilizando los atributos .real y .imag, respectivamente:

Ejemplo

z_parte_real = z.real
z_parte_imag = z.imag

print(f"La parte real de z es: {z_parte_real} y la parte imaginaria es: {z_parte_imag}")

La parte real de z es: 1.0 y la parte imaginaria es: 2.0

Raíces de un número complejo#

Si

\[ z=r(\cos\theta+i\sin\theta) \]

entonces las \(n\) raíces de \(z\) están dadas por

\[ z_k=\sqrt[n]{r}\left(\cos\left(\frac{\theta+2k\pi}{n}\right) +i\sin\left(\frac{\theta+2k\pi}{n}\right)\right) \qquad k=0,1,\ldots,n-1. \]

Ejemplo

Determine las raíces cúbicas de la unidad

\[ z=1 \]

i.e. todos los números complejos \(w\) tales que

\[ w^3=1 \]
# número complejo

z = 1 + 0j

r = np.abs(z)
theta = np.angle(z)

n = 3

for k in range(n):
    phi = (theta + 2*np.pi*k)/n
    rho = r**(1/n)

    z_k = rho*(np.cos(phi) + 1j*np.sin(phi))

    print(f"Raíz {k + 1}:")
    print("Forma rectangular:", z_k)
    print(f"Forma polar: {rho}(Cos({phi:.3f} + i Sin({phi:.3f})))")

Forma rectangular: (1+0j)
Forma polar: 1.0(Cos(0.000 + i Sin(0.000)))

Ejercicio#

Determine las raíces cuartas de la unidad

z = 1 + 0j
r = np.abs(z)
theta = np.angle(z)
n = 4
for k in range(n):
  phi = (theta + 2*k*np.pi)/n
  rho = r**(1/n)
  z_k = rho*(np.cos(phi)+1j*np.sin(phi))

  print()
