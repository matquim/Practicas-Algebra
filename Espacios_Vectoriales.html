1. Base y dimensión#

Ejemplo de Base y dimensión en \(\mathbb{R}^3\)#

  1. Verificar si los vectores \(\mathbf{u}\), \(\mathbf{v}\) y \(\mathbf{w}\) forman una base de \(\mathbb{R}^3\)

\[\begin{split} \mathbf{u}= \begin{pmatrix} 1\\ 2\\ 0 \end{pmatrix}, \qquad \mathbf{v}= \begin{pmatrix} 0\\ 1\\ 1 \end{pmatrix}, \qquad \mathbf{w}= \begin{pmatrix} 2\\ 0\\ 1 \end{pmatrix} \end{split}\]

Usando Python podemos hacer los siguiente

  • Construir una matriz \(A\) cuyas columnas sean los vectores \(\mathbf{u}\), \(\mathbf{v}\) y \(\mathbf{w}\)

  • Usar Python para calcular el determinante de \(A\) y determinar si los vectores efectivamente forman una base del espacio \(\mathbb{R}^3\)

\(\mathbf{u}\), \(\mathbf{v}\) y \(\mathbf{w}\) son linealmente independientes y por lo tanto forman una base de \(\mathbb{R}^3\)

  1. Sea \( \mathbf{p}= \begin{pmatrix} 3\\ 4\\ 2 \end{pmatrix} \)

Exprese el vector \(\mathbf{p}\) como una combinación lineal de los vectores \(\mathbf{u}\), \(\mathbf{v}\) y \(\mathbf{w}\), es decir

\( \mathbf{p} = \alpha_1\mathbf{u} + \alpha_2\mathbf{v} + \alpha_3\mathbf{w}\)

Ejemplos de espacios generados#

  1. Considere el vector

\[\begin{split} \mathbf{v_1}= \begin{pmatrix} 2\\ 1 \end{pmatrix} \end{split}\]

El espacio generado por \(\mathbf{v_1}\) está dado por

\[ \operatorname{gen}\left\{\mathbf{v_1} \right\} = \left\{\mathbf{v}: \mathbf{v} = a\mathbf{v_1}, a\in\mathbb{R} \right\} \]

  1. Considere los vectores

\[\begin{split} \mathbf{w_1}= \begin{pmatrix} 1\\ 0 \end{pmatrix}, \qquad \mathbf{w_2}= \begin{pmatrix} 0\\ 1 \end{pmatrix} \end{split}\]

El espacio generado por \(\mathbf{w_1}\) y \(\mathbf{w_2}\) está dado por

\[ \operatorname{gen}\left\{\mathbf{w_1},\mathbf{w_2}\right\} = \left\{ \mathbf{w}: \mathbf{w}=a\mathbf{w_1}+b\mathbf{w_2}, \;a,b\in\mathbb{R} \right\}, \]

en donde

\[\begin{split} a\mathbf{w_1}+b\mathbf{w_2} = a \begin{pmatrix} 1\\ 0 \end{pmatrix} + b \begin{pmatrix} 0\\ 1 \end{pmatrix} = \begin{pmatrix} a\\ b \end{pmatrix} \end{split}\]

Considere los vectores

\[\begin{split} \mathbf{w_1}= \begin{pmatrix} 1\\ 0\\ 0 \end{pmatrix}, \qquad \mathbf{w_2}= \begin{pmatrix} 0\\ 1\\ 1 \end{pmatrix}. \end{split}\]

El espacio generado por \(\mathbf{w_1}\) y \(\mathbf{w_2}\) está dado por

\[ \operatorname{gen}\left\{\mathbf{w_1},\mathbf{w_2}\right\} = \left\{ \mathbf{w}: \mathbf{w}=a\mathbf{w_1}+b\mathbf{w_2}, \;a,b\in\mathbb{R} \right\}. \]

Representación gráfica de complejos#

Raíces de un número complejo#

Si

\[ z=r(\cos\theta+i\sin\theta) \]

entonces las \(n\) raíces de \(z\) están dadas por

\[ z_k=\sqrt[n]{r}\left(\cos\left(\frac{\theta+2k\pi}{n}\right) +i\sin\left(\frac{\theta+2k\pi}{n}\right)\right) \qquad k=0,1,\ldots,n-1. \]

Ejemplo
